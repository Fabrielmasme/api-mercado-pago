# Mercado Pago API - .NET

Proyecto backend en ASP.NET Core que implementa pagos utilizando la API de Mercado Pago.

## Tecnologías
- ASP.NET Core
- C#
- Entity Framework Core
- Mercado Pago SDK
- SQL Server

## Funcionalidades
- Creación de preferencias de pago
- Integración con checkout de Mercado Pago
- Manejo de respuestas y estados de pago

## Objetivo
Proyecto personal y demostrativo para mostrar integración de pasarelas de pago en .NET.
